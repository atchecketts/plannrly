## Notes

### CSS Utilities to Include

```css
/* Safe area insets */
.pb-safe { padding-bottom: env(safe-area-inset-bottom); }
.pt-safe { padding-top: env(safe-area-inset-top); }

/* Touch utilities */
.touch-manipulation { touch-action: manipulation; }
.touch-pan-x { touch-action: pan-x; }
.touch-pan-y { touch-action: pan-y; }

/* Minimum tap targets */
.tap-target { min-width: 44px; min-height: 44px; }

/* iOS momentum scrolling */
.scroll-momentum { -webkit-overflow-scrolling: touch; }

/* Prevent text selection during swipes */
.no-select { user-select: none; }
```

### Performance Optimizations
- Use `loading="lazy"` on images below the fold
- Consider code splitting for mobile-only features
- Use appropriate image formats (WebP with fallback)
- Minimize render-blocking CSS
