# UAT: Feature 10.6 - Document & Certification Management (Premium)

## Test Environment Setup

**Prerequisites:**
- Admin user with premium subscription
- Employee accounts
- Test documents (PDF, images) for upload
- Roles requiring certifications

**Test URL:** `/documents`, `/certifications`

---

## Test Cases

### TC-10.6.1: Feature Gate - Premium Required

**Steps:**
1. Log in as admin without premium subscription
2. Try to access document management

**Expected Results:**
- [ ] Access denied or upgrade prompt shown
- [ ] Feature marked as premium
- [ ] Basic employee management still works

---

### TC-10.6.2: View Documents Index

**Steps:**
1. Log in as admin with premium
2. Navigate to Documents
3. View document list

**Expected Results:**
- [ ] Documents list loads
- [ ] Shows uploaded documents
- [ ] Categorization visible
- [ ] Upload button available

---

### TC-10.6.3: Upload Employee Document

**Steps:**
1. Click Upload Document
2. Select employee
3. Choose document type (e.g., ID, Contract)
4. Upload file
5. Save

**Expected Results:**
- [ ] File selector works
- [ ] Can select employee
- [ ] Can categorize document type
- [ ] Document uploads successfully
- [ ] Appears in document list

---

### TC-10.6.4: Document Type Categorization

**Steps:**
1. Upload multiple documents with different types:
   - ID/Passport
   - Employment contract
   - Emergency contact form
   - Other
2. View categorized list

**Expected Results:**
- [ ] Multiple document types available
- [ ] Documents appear in correct category
- [ ] Can filter by document type
- [ ] Custom categories possible

---

### TC-10.6.5: View/Download Document

**Steps:**
1. Find an uploaded document
2. Click to view
3. Download document

**Expected Results:**
- [ ] Document preview loads (if supported type)
- [ ] Download option available
- [ ] File downloads correctly
- [ ] Original file integrity maintained

### TC-10.6.6: Define Certification Types

**Steps:**
1. Navigate to Certifications
2. Create new certification type:
   - Name: "Food Safety Certificate"
   - Validity period: 2 years
3. Save

**Expected Results:**
- [ ] Can create certification types
- [ ] Name and validity period configurable
- [ ] Certification type appears in list
- [ ] Can edit/delete certification types

---

### TC-10.6.7: Track Employee Certification

**Steps:**
1. Navigate to employee's certifications
2. Add certification:
   - Select certification type
   - Set issue date
   - Set expiry date
   - Upload certificate document
3. Save

**Expected Results:**
- [ ] Can add certification to employee
- [ ] Dates captured correctly
- [ ] Document attached
- [ ] Certification shows in employee profile

---

### TC-10.6.8: Define Required Certifications per Role

**Steps:**
1. Navigate to role settings (or certifications)
2. Assign required certifications to role:
   - "Server" requires "Food Safety Certificate"
3. Save

**Expected Results:**
- [ ] Can link certifications to roles
- [ ] Role shows required certifications
- [ ] Employees in role checked against requirements
