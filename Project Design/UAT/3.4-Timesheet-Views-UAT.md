# UAT: Feature 3.4 - Timesheet Views

## Test Environment Setup

**Prerequisites:**
- One admin user
- Multiple employee users with time entries
- Completed shifts with time tracking
- Data spanning multiple weeks
- Various departments and locations

**Test URLs:**
- Admin Timesheets: `/timesheets`
- Employee Timesheet: `/timesheets/me`
- Weekly View: `/timesheets/weekly`

---

## Test Cases

### TC-3.4.1: Admin View All Timesheets

**Steps:**
1. Log in as an admin
2. Navigate to Timesheets in sidebar

**Expected Results:**
- [ ] Timesheet index loads
- [ ] Shows all employees' time data
- [ ] Grouped by employee or date
- [ ] Summary totals visible

---

### TC-3.4.2: Employee View Own Timesheet

**Steps:**
1. Log in as an employee
2. Navigate to "My Timesheet" or Timesheets

**Expected Results:**
- [ ] Shows only own time entries
- [ ] Cannot see other employees
- [ ] Weekly view of hours
- [ ] Clear summary of worked hours

---

### TC-3.4.3: Weekly Timesheet View

**Steps:**
1. Log in as admin or employee
2. Select weekly view
3. Choose specific week

**Expected Results:**
- [ ] Shows Mon-Sun or Sun-Sat (configurable)
- [ ] Each day shows scheduled vs actual
- [ ] Total hours per day
- [ ] Weekly total at end

---

### TC-3.4.4: Side-by-Side Scheduled vs Actual
