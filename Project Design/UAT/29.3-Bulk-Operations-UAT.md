# UAT: Feature 29.3 - Bulk Operations

## Test Environment Setup

**Prerequisites:**
- Admin user with full permissions
- Multiple employees in the system
- Multiple shifts (published and draft)
- Multiple pending leave requests
- Multiple pending timesheet entries
- Shift templates configured

**Test URL:** `/shifts`, `/employees`, `/leave`, `/timesheets`

---

## Test Cases

### TC-29.3.1: Select Multiple Shifts

**Steps:**
1. Log in as admin
2. Navigate to schedule/shift view
3. Click checkbox on first shift
4. Click checkbox on additional shifts
5. Observe selection state

**Expected Results:**
- [ ] Individual shift checkboxes available
- [ ] Multiple shifts can be selected
- [ ] Selection count is displayed
- [ ] "Select All" option available
- [ ] Bulk action bar appears

---

### TC-29.3.2: Bulk Create Shifts from Template

**Steps:**
1. Navigate to shift creation
2. Find "Create from Template" option
3. Select a shift template
4. Select date range (e.g., next week)
5. Select employees to assign
6. Generate shifts

**Expected Results:**
- [ ] Template selection available
- [ ] Date range can be specified
- [ ] Can assign multiple employees
- [ ] Shifts are created for all days
- [ ] Confirmation shows count created
