# UAT: Feature 10.3 - Labor Demand Forecasting (Premium)

## Test Environment Setup

**Prerequisites:**
- Admin user with premium subscription
- Minimum 3 months of historical scheduling data
- Various locations and departments
- Clock-in/out data for historical periods

**Test URL:** `/forecasting`

---

## Test Cases

### TC-10.3.1: Feature Gate - Premium Required

**Steps:**
1. Log in as admin without premium subscription
2. Try to access labor forecasting

**Expected Results:**
- [ ] Access denied or upgrade prompt shown
- [ ] Feature marked as premium
- [ ] Standard scheduling still works

---

### TC-10.3.2: View Forecast Dashboard

**Steps:**
1. Log in as admin with premium
2. Navigate to Forecasting
3. View forecast dashboard

**Expected Results:**
- [ ] Dashboard loads successfully
- [ ] Shows forecast summary
- [ ] Chart/graph visualization present
- [ ] Date range selector available

---

### TC-10.3.3: Daily Staffing Predictions

**Steps:**
1. Navigate to forecasting
