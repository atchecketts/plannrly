# UAT: Feature 11.2 - Pricing Page

## Test Environment Setup

**Prerequisites:**
- Public access (no authentication)
- Multiple devices for responsive testing
- Landing page completed (shared layout)

**Test URLs:**
- Pricing Page: `/pricing`
- Landing Page: `/` (for navigation testing)

---

## Test Cases

### TC-11.2.1: Pricing Page Loads Successfully

**Steps:**
1. Navigate to `/pricing`

**Expected Results:**
- [ ] Page loads without errors
- [ ] No authentication required
- [ ] All pricing content visible
- [ ] Page uses public layout

---

### TC-11.2.2: All Pricing Tiers Displayed

**Steps:**
1. Navigate to pricing page
2. Review pricing tier cards

**Expected Results:**
- [ ] Starter tier visible
- [ ] Professional tier visible
- [ ] Enterprise tier visible
- [ ] Each tier shows name
- [ ] Each tier shows price
- [ ] Each tier lists key features

---

### TC-11.2.3: Monthly/Annual Toggle

**Steps:**
1. Navigate to pricing page
2. Locate the billing period toggle
3. Toggle from Monthly to Annual
4. Toggle back to Monthly

**Expected Results:**
- [ ] Toggle is clearly visible
- [ ] Default state is clear (Monthly or Annual)
- [ ] Prices update when toggled
- [ ] Annual prices show discounted rate
- [ ] Toggle animation is smooth

---

### TC-11.2.4: Annual Savings Display

- [ ] Savings clearly visible
