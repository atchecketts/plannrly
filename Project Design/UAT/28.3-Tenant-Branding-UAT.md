# UAT: Feature 28.3 - Tenant Branding

## Test Environment Setup

**Prerequisites:**
- Admin user for tenant on Starter tier
- Admin user for tenant on Professional tier
- Admin user for tenant on Enterprise tier
- Sample logo images (PNG, JPG, SVG)
- Knowledge of subscription tier restrictions
- Access to custom domain setup (Enterprise)

**Test URL:** `/settings/branding`

---

## Test Cases

### TC-28.3.1: Access Branding Settings

**Steps:**
1. Log in as tenant admin
2. Navigate to Settings
3. Click on "Branding" or "Appearance"

**Expected Results:**
- [ ] Branding settings page loads
- [ ] Shows available customization options
- [ ] Options may be limited by subscription tier
- [ ] Current branding (if any) is displayed

---

### TC-28.3.2: Upload Logo (All Tiers)

**Steps:**
1. Navigate to branding settings
2. Click "Upload Logo"
3. Select a PNG image file
4. Preview the logo
5. Save changes

**Expected Results:**
- [ ] File upload interface works
- [ ] Can preview logo before saving
- [ ] Logo saves successfully
- [ ] Logo appears in application header
- [ ] Available on all subscription tiers

---

### TC-28.3.3: Logo Format Validation

**Steps:**
1. Try uploading various file formats
2. Upload PNG, JPG, SVG
3. Try uploading invalid format (PDF, DOC)

**Expected Results:**
- [ ] PNG uploads successfully
- [ ] JPG uploads successfully
- [ ] SVG uploads successfully (if supported)
- [ ] Invalid formats show error message
- [ ] Error message is helpful

---

### TC-28.3.4: Logo Size Restrictions

**Steps:**
1. Try uploading very large image (5MB+)
2. Try uploading very small image
3. Try uploading image with wrong dimensions

**Expected Results:**
- [ ] Large files show size limit error
- [ ] Minimum dimensions enforced
- [ ] Maximum dimensions enforced
- [ ] Helpful error messages shown
- [ ] Suggested dimensions displayed

---

### TC-28.3.5: Logo Display Across Application

**Steps:**
1. Upload a logo
2. Navigate through different pages
3. View login page (if applicable)
4. View employee dashboard

**Expected Results:**
- [ ] Logo appears in navigation header
- [ ] Logo appears on login page
- [ ] Logo maintains quality across pages
- [ ] Logo properly sized in all locations

---

### TC-28.3.6: Color Customization (Professional+ Only)

**Steps:**
1. Log in as Professional tier admin
2. Navigate to branding settings
3. Find color customization section
4. Change primary color
5. Change secondary/accent color
6. Save changes

**Expected Results:**
- [ ] Color options available for Professional tier
- [ ] Color picker or hex input available
- [ ] Can preview colors before saving
- [ ] Colors save successfully
- [ ] Not available on Starter tier

---

### TC-28.3.7: Color Application in UI

**Steps:**
1. Set custom primary color (e.g., brand blue)
2. Navigate through the application
3. Check buttons, headers, links

**Expected Results:**
- [ ] Primary color applied to main buttons
- [ ] Primary color applied to headers

### TC-28.3.8: Starter Tier Color Restriction

**Steps:**
1. Log in as Starter tier admin

### TC-28.3.9: Custom Domain Setup (Enterprise Only)

- [ ] Domain verification process works
- [ ] Domain saves after verification
