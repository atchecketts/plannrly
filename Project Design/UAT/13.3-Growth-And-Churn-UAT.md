# UAT: Feature 13.3 - Growth & Churn Reports

## Test Environment Setup

**Prerequisites:**
- SuperAdmin user account
- Historical subscription data (at least 6 months)
- Data showing new subscriptions, upgrades, downgrades
- Data showing cancelled subscriptions (churn)
- Reactivated subscriptions

**Test URL:** `/super-admin/analytics/growth`, `/super-admin/analytics/churn`

---

## Test Cases

### TC-13.3.1: Access Growth Reports (SuperAdmin)

**Steps:**
1. Log in as a SuperAdmin
2. Navigate to "Analytics" in the super-admin sidebar
3. Click on "Growth Reports"

**Expected Results:**
- [ ] Page loads successfully
- [ ] Growth metrics dashboard displays
- [ ] Charts render correctly
- [ ] Time period selector is visible

---

### TC-13.3.2: New Subscriptions Metric

**Steps:**
1. Log in as SuperAdmin

3. View new subscriptions count for current period

**Expected Results:**
