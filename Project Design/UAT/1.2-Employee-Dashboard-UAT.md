# UAT: Feature 1.2 - Employee Dashboard

## Test Environment Setup

**Prerequisites:**
- One employee user with assigned shifts
- Published shifts for the next 7 days
- At least one pending leave request
- At least one active shift swap request
- One admin user (for comparison)

**Test URL:** `/dashboard`

---

## Test Cases

### TC-1.2.1: Employee Sees Employee Dashboard

**Steps:**
1. Log in as an employee
2. You should be automatically redirected to the dashboard

**Expected Results:**
- [ ] Dashboard loads successfully
- [ ] Shows personalized content for the employee
- [ ] Layout is different from admin dashboard

---

### TC-1.2.2: Admin Sees Admin Dashboard

**Steps:**
1. Log in as an admin
2. Navigate to dashboard

**Expected Results:**
- [ ] Shows admin dashboard (not employee dashboard)
- [ ] Shows organization-wide statistics
- [ ] Different layout from employee view

---

### TC-1.2.3: Next Shift Countdown

**Steps:**
1. Log in as an employee with an upcoming published shift
2. View the dashboard

**Expected Results:**
- [ ] "Next Shift" section is visible
- [ ] Shows countdown timer (days, hours, minutes, or "Starting soon")
- [ ] Shows shift date, time, location, and role
- [ ] Timer updates in real-time (without page refresh)

---

### TC-1.2.4: Upcoming Shifts Display

**Steps:**
1. Log in as an employee with multiple shifts in the next 7 days
2. View the "Upcoming Shifts" section

**Expected Results:**
- [ ] Shows shifts for the next 7 days
- [ ] Each shift shows date, day name, start/end time
- [ ] Shows business role with color indicator
- [ ] Shows department and location
- [ ] Shifts are ordered by date

---

### TC-1.2.5: No Upcoming Shifts

**Steps:**
1. Log in as an employee with NO shifts in the next 7 days
2. View the dashboard

**Expected Results:**
- [ ] Shows friendly empty state message
- [ ] No errors displayed
- [ ] Suggests checking the full schedule

---

### TC-1.2.6: Leave Requests Section

**Steps:**
1. Log in as an employee with leave requests
2. View the "My Leave Requests" section

**Expected Results:**
- [ ] Shows recent leave requests
- [ ] Displays leave type, dates, and status
- [ ] Status badges show correct colors (Draft, Requested, Approved, Rejected)
- [ ] Shows total days for each request

---

### TC-1.2.7: Shift Swaps Section

**Steps:**
1. Log in as an employee with swap requests
2. View the "My Shift Swaps" section

**Expected Results:**
- [ ] Shows swap requests where you are requester or target
- [ ] Shows other employee's name
- [ ] Shows shift date being swapped
- [ ] Shows current status

---

### TC-1.2.8: Quick Actions

**Steps:**
1. Log in as an employee
2. Find the quick actions section
3. Click "Request Leave"
4. Go back and click "View Schedule"

**Expected Results:**
- [ ] Quick actions are visible
- [ ] "Request Leave" navigates to leave request form
- [ ] "View Schedule" navigates to schedule page
- [ ] Buttons are clearly labeled

---

### TC-1.2.9: Statistics Display

**Steps:**
1. Log in as an employee with shifts this week
2. View the statistics/summary section

**Expected Results:**
- [ ] Shows "Shifts This Week" count
- [ ] Shows "Hours This Week" total
- [ ] Numbers are accurate

---

### TC-1.2.10: Responsive Design

**Steps:**
1. Log in as an employee
2. View dashboard on desktop
3. Resize browser to tablet size
4. Resize to mobile size

**Expected Results:**
- [ ] Layout adjusts appropriately at each breakpoint
- [ ] All content remains accessible
- [ ] No horizontal scrolling on mobile
- [ ] Countdown timer is readable on all sizes

---

### TC-1.2.11: Real-time Countdown

**Steps:**
1. Log in as an employee with a shift starting within the hour
2. Watch the countdown timer for 2 minutes

**Expected Results:**
- [ ] Timer counts down in real-time
- [ ] Minutes and seconds update
- [ ] No page refresh required
- [ ] When shift starts, shows "Starting now" or similar
