# UAT: Feature 10.8 - Custom Branding / White Label (Enterprise)

## Test Environment Setup

**Prerequisites:**
- Admin user with enterprise subscription
- Test logo images (various sizes)
- Test favicon
- Color palette to test

**Test URL:** `/branding`, `/settings/branding`

---

## Test Cases

### TC-10.8.1: Feature Gate - Enterprise Required

**Steps:**
1. Log in as admin with premium (not enterprise)
2. Try to access branding settings

**Expected Results:**
- [ ] Access denied or upgrade prompt shown
- [ ] Feature marked as Enterprise
- [ ] Default branding remains

---

### TC-10.8.2: View Branding Settings

**Steps:**
1. Log in as admin with enterprise
2. Navigate to Branding settings
3. View customization options

**Expected Results:**
- [ ] Branding page loads
- [ ] Shows current branding (default or custom)
- [ ] All customization options visible:
  - Logo
  - Colors
  - Favicon
  - Login page
