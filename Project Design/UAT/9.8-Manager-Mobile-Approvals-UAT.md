# UAT: Feature 9.8 - Manager Mobile Approvals

## Test Environment Setup

**Prerequisites:**
- Manager/admin user account
- Employee user accounts
- Pending leave requests
- Pending shift swap requests
- Open shifts requiring assignment
- Employees currently clocked in

**Test URL:** `/approvals`, `/dashboard` (manager view)

---

## Test Cases

### TC-9.8.1: Mobile Pending Approvals List

**Steps:**
1. Log in as a manager on mobile device
2. Navigate to Approvals or approval section

**Expected Results:**
- [ ] Pending approvals list displays
- [ ] Shows leave requests awaiting approval
- [ ] Shows swap requests awaiting approval
- [ ] Count of pending items visible
- [ ] Mobile-optimized card layout

---

### TC-9.8.2: Quick Approve Action

**Steps:**
1. On pending approvals list
2. Find approve button/action for an item
3. Tap to approve

**Expected Results:**
- [ ] One-tap approve works
- [ ] Confirmation shown (or immediate success)
- [ ] Item removed from pending list
- [ ] Status updates to approved

---

### TC-9.8.3: Quick Deny Action

**Steps:**
1. On pending approvals list
2. Find deny button/action for an item
3. Tap to deny

**Expected Results:**
- [ ] One-tap deny works
- [ ] Optional: reason prompt appears
- [ ] Item removed from pending list
- [ ] Status updates to denied

---

### TC-9.8.4: Bulk Approval Functionality

**Steps:**
1. On pending approvals list
2. Select multiple items (checkbox or long-press)
3. Use bulk approve action

**Expected Results:**
- [ ] Can select multiple items
- [ ] Bulk approve button appears
- [ ] All selected items approved
- [ ] Success confirmation for bulk action

---

### TC-9.8.5: Employee Contact Quick Actions

**Steps:**
1. On a leave request or approval item
2. Find employee contact options
3. Tap call or message button

**Expected Results:**
- [ ] Phone icon initiates call (opens dialer)
- [ ] Message icon opens SMS or messaging app
- [ ] Employee's contact info used correctly
- [ ] Quick access without navigating to profile

### TC-9.8.6: Today's Coverage At-a-Glance

### TC-9.8.8: Assign Open Shift

**Steps:**
1. Find an open/unfilled shift
