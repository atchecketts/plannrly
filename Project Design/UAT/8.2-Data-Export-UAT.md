# UAT: Feature 8.2 - Data Export

## Test Environment Setup

**Prerequisites:**
- Admin user with full access
- Multiple employees with schedule data
- Shifts spanning several weeks
- Timesheet/clock data available
- Leave requests with various statuses
- Employees in different departments and locations

**Test URL:** Various pages with export buttons

---

## Test Cases

### TC-8.2.1: Export Schedule to CSV

**Steps:**
1. Log in as an admin
2. Navigate to Schedule view
3. Select a date range (e.g., current week)
4. Click "Export" > "CSV"

**Expected Results:**
- [ ] CSV file downloads
- [ ] Contains all shifts in selected range
- [ ] Includes: date, time, employee, role, department, location
- [ ] File opens correctly in spreadsheet software

---
