# UAT: Feature 21.4 - Professional Services

## Test Environment Setup

**Prerequisites:**
- Tenant with admin access
- Email access to verify routing
- Admin access to view service requests
- Test scenarios for each service type

**Test URL:** `/services`, `/settings/services`

---

## Test Cases

### TC-21.4.1: View Professional Services Page

**Steps:**
1. Log in as any user
2. Navigate to Professional Services page

**Expected Results:**
- [ ] Services page loads successfully
- [ ] List of available services displayed
- [ ] Each service has description
- [ ] Pricing or "contact for quote" shown

---

### TC-21.4.2: Service Offerings Displayed

**Steps:**
1. View the Professional Services page
2. Review each service offering

**Expected Results:**
- [ ] Migration Assistance service listed
- [ ] Training Session service listed
- [ ] Any other services clearly shown
- [ ] Benefits of each service explained

---

### TC-21.4.3: Request Migration Assistance Form

**Steps:**
1. Navigate to Professional Services
2. Click "Request Migration Assistance"
3. Fill in the form:
   - Current system being migrated from
   - Data volume estimate
   - Preferred timeline
   - Contact information
4. Submit the form

**Expected Results:**
- [ ] Form loads correctly
- [ ] All fields validate
- [ ] Submission succeeds
- [ ] Confirmation message shown
- [ ] Confirmation email sent

---

### TC-21.4.4: Request Training Session Form

**Steps:**
1. Navigate to Professional Services
2. Click "Request Training Session"
3. Fill in the form:
   - Training topic/area
   - Number of attendees
   - Preferred dates/times
   - Contact information
4. Submit the form

**Expected Results:**
- [ ] Form loads correctly
- [ ] All fields validate
- [ ] Submission succeeds
- [ ] Confirmation message shown
- [ ] Confirmation email sent

---

### TC-21.4.5: Form Validation - Required Fields

**Steps:**
1. Open any service request form
2. Try to submit without filling required fields

**Expected Results:**
- [ ] Validation errors shown
- [ ] Required fields highlighted
- [ ] Cannot submit incomplete form
- [ ] Clear error messages

---

### TC-21.4.6: Inquiry Routing to Sales Team

**Steps:**
1. Submit a service request
2. Check sales team inbox/notification system

**Expected Results:**
- [ ] Sales team receives notification
- [ ] Email contains all submitted details
- [ ] Includes tenant/company information
- [ ] Reply-to set correctly

---

### TC-21.4.7: Service Request Tracking in Admin Panel

**Steps:**
1. Log in as platform admin
2. Navigate to Admin > Service Requests

**Expected Results:**
- [ ] All service requests listed
- [ ] Shows request type, tenant, date
- [ ] Status tracking (new, in progress, completed)
- [ ] Can view full request details

---

### TC-21.4.8: Update Service Request Status (Admin)

**Steps:**
1. Find a service request in admin panel
2. Update its status (e.g., "In Progress", "Completed")
3. Save changes

**Expected Results:**
- [ ] Status updates successfully
- [ ] History/audit of status changes
- [ ] Tenant may receive notification
- [ ] Admin notes can be added

---

### TC-21.4.9: Tenant View of Their Service Requests

**Steps:**
1. Log in as the tenant who submitted a request
2. Navigate to see their service request status

**Expected Results:**
- [ ] Can see submitted requests
- [ ] Shows current status
- [ ] Contact information for follow-up
- [ ] Any updates or responses visible

---

### TC-21.4.10: Service Request Confirmation Email

**Steps:**
1. Submit a service request
2. Check the tenant's email inbox

**Expected Results:**
- [ ] Confirmation email received
- [ ] Details of request summarized
- [ ] Expected response time mentioned
- [ ] Contact information for questions

---

### TC-21.4.11: Service Page Access Without Login

**Steps:**
1. Log out of the application
2. Navigate to the Professional Services page

**Expected Results:**
- [ ] Services page is publicly accessible
- [ ] Can view service offerings
- [ ] Prompted to log in to request

### TC-21.4.12: Custom Service Inquiry

**Steps:**

**Expected Results:**
- [ ] Custom inquiry option available

### TC-21.4.13: Mobile Responsiveness

**Steps:**
