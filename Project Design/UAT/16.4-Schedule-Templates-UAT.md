# UAT: Feature 16.4 - Schedule Templates

## Test Environment Setup

**Prerequisites:**
- Admin user account
- Published schedule with shifts
- Multiple employees with different roles
- Different locations/departments

**Test URLs:**
- `/schedule-templates`
- `/schedule-templates/create`

---

## Test Cases

### TC-16.4.1: Access Schedule Templates (Admin)

**Steps:**
1. Log in as an admin
2. Navigate to "Schedule Templates" in the sidebar

**Expected Results:**
- [ ] Templates page loads successfully
- [ ] List of existing templates shown (or empty state)
- [ ] Create new template button visible
- [ ] Template categories visible

---

### TC-16.4.2: Employee Cannot Access Templates

**Steps:**
1. Log in as an employee
2. Try to navigate to `/schedule-templates`

**Expected Results:**
- [ ] Access denied (403 error)
- [ ] Templates not visible in sidebar
- [ ] No template data exposed

---

### TC-16.4.3: Save Current Schedule as Template

**Steps:**
1. Log in as admin
2. Navigate to schedule view with shifts
3. Click "Save as Template"
4. Enter template name and description
5. Save

**Expected Results:**
- [ ] Save as template option available
- [ ] Template creation form shown
- [ ] Template saved successfully
- [ ] Appears in templates list

---

### TC-16.4.4: Create Template Manually

**Steps:**
1. Navigate to schedule templates
2. Click "Create Template"
3. Add shifts to the template
4. Configure shift times and roles
5. Save template

**Expected Results:**
- [ ] Manual template creation works
- [ ] Can add multiple shifts
- [ ] Shift details configurable
- [ ] Template saves correctly

---

### TC-16.4.5: View Template Library

**Steps:**
1. Navigate to schedule templates
2. Browse available templates

**Expected Results:**
- [ ] All templates listed
- [ ] Shows template name and description
- [ ] Shows number of shifts in template
- [ ] Shows category/type

---

### TC-16.4.6: Template Categories

**Steps:**
1. Create templates with different categories
2. Navigate to template library
3. Filter by category

**Expected Results:**
- [ ] Categories available (Standard, Seasonal, etc.)

- [ ] Templates organized clearly

### TC-16.4.7: Apply Template to Future Week

**Steps:**
1. Navigate to schedule templates
2. Select a template
3. Click "Apply to Schedule"
4. Select target week
5. Confirm application

**Expected Results:**
- [ ] Week picker shown
- [ ] Template is applied
- [ ] Shifts created based on template
- [ ] Dates adjusted to target week

---

### TC-16.4.8: Smart Employee Matching by Role

**Steps:**
1. Apply a template with role-specific shifts
2. Observe employee assignment

**Expected Results:**
- [ ] Employees matched by business role
- [ ] Qualified employees assigned
- [ ] Unassigned shifts if no match
- [ ] Clear indication of assignment status

---

### TC-16.4.9: Preview Before Applying
