# UAT: Feature 6.4 - Leave Calendar View

## Test Environment Setup

**Prerequisites:**
- One admin user with full access
- Multiple employees in different departments and locations
- Various leave requests with different statuses (Pending, Approved, Rejected)
- Multiple leave types configured (Annual, Sick, Personal, etc.)
- Leave requests spanning multiple days

**Test URL:** `/leave/calendar`

---

## Test Cases

### TC-6.4.1: Access Leave Calendar (Admin)

**Steps:**
1. Log in as an admin
2. Navigate to Leave Management in the sidebar
3. Click on "Calendar View" or navigate to `/leave/calendar`

**Expected Results:**
- [ ] Leave calendar page loads successfully
- [ ] Shows current month by default
- [ ] Leave requests are displayed on calendar
- [ ] Navigation options are available

---

### TC-6.4.2: Access Leave Calendar (Employee)

**Steps:**
1. Log in as an employee
2. Navigate to Leave section
3. Look for calendar view option

**Expected Results:**
- [ ] Employee can access leave calendar
- [ ] Shows team/department leave (based on permissions)
- [ ] Personal leave is highlighted
- [ ] May be limited to own department

---

### TC-6.4.3: Color Coding by Leave Type

**Steps:**
1. Log in as an admin
2. Navigate to leave calendar
3. View leave requests of different types

**Expected Results:**
- [ ] Annual leave shows in one color (e.g., blue)
- [ ] Sick leave shows in different color (e.g., red)
- [ ] Personal leave has its own color
- [ ] Custom leave types have assigned colors
- [ ] Legend shows color meanings

---

### TC-6.4.4: Status Indicators

**Steps:**
1. Log in as an admin
2. Navigate to leave calendar
3. View leave requests with different statuses
