# UAT: Feature 16.5 - Smart Fill (Auto-Scheduling Assistant)

## Test Environment Setup

**Prerequisites:**
- Admin user account
- Multiple employees with different roles and availability
- Unassigned shifts in the schedule
- Employee target hours configured
- Employee availability set

**Test URL:** Schedule view with unassigned shifts

---

## Test Cases

### TC-16.5.1: Access Smart Fill Feature

**Steps:**
1. Log in as admin
2. Navigate to schedule view
3. Locate "Smart Fill" button

**Expected Results:**
- [ ] Smart Fill button is visible
- [ ] Located in schedule toolbar or actions
- [ ] Only visible to admin/manager

---

### TC-16.5.2: Initiate Smart Fill

**Steps:**
1. Have unassigned shifts in the schedule
2. Click "Smart Fill"

**Expected Results:**
- [ ] Smart Fill process starts
- [ ] Loading/processing indicator shown
- [ ] Suggestions are generated
- [ ] Preview mode is entered

---

### TC-16.5.3: View Suggestions

**Steps:**
1. Run Smart Fill
2. View the generated suggestions

**Expected Results:**
- [ ] Each unassigned shift shows suggestion
- [ ] Suggested employee name visible
- [ ] Explanation/score for suggestion
- [ ] Alternative candidates shown

---

### TC-16.5.4: Candidates Based on Availability

**Steps:**
1. Set employee as unavailable Monday
2. Run Smart Fill on Monday shift
3. Check suggestions

**Expected Results:**
- [ ] Unavailable employee not suggested
- [ ] Only available employees considered
- [ ] Availability is primary filter

---

### TC-16.5.5: Candidates Based on Qualifications

**Steps:**
1. Create shift requiring specific role
2. Run Smart Fill
3. Check suggestions

**Expected Results:**
- [ ] Only qualified employees suggested
- [ ] Role match is required
- [ ] Shows qualification match

---

### TC-16.5.6: Ranking by Fairness Score

**Steps:**
1. Run Smart Fill on multiple shifts
2. Review suggested employee ranking

**Expected Results:**
- [ ] Employees ranked by fairness
- [ ] Hours distribution considered
- [ ] No single employee overloaded
- [ ] Fairness score shown

---

### TC-16.5.7: Respect Target Hours

**Steps:**
1. Set target hours for employees
2. Run Smart Fill
3. Review suggestions

**Expected Results:**
- [ ] Target hours considered
- [ ] Employees near target deprioritized
- [ ] Employees under target prioritized
- [ ] Target shown in suggestions

---

### TC-16.5.8: Overtime Limit Warnings

**Steps:**
1. Have employee near overtime limit
2. Run Smart Fill
3. Check if warning appears

**Expected Results:**
- [ ] Overtime warning shown
- [ ] Can still assign if desired
- [ ] Warning is prominent
- [ ] Overtime hours displayed

---

### TC-16.5.9: Consider Recent Shifts for Fairness

**Steps:**
1. Have employee with many recent shifts
2. Run Smart Fill
3. Check suggestions

**Expected Results:**
- [ ] Recent workload considered
- [ ] Overworked employees deprioritized
- [ ] Balance across team

---

### TC-16.5.10: View Alternative Suggestions

**Steps:**
1. Run Smart Fill
2. For a shift, click to view alternatives

**Expected Results:**
- [ ] Multiple candidates shown
- [ ] Ranked by suitability
- [ ] Can select alternative
- [ ] Reasons for ranking shown

---

### TC-16.5.11: Preview Before Applying

**Steps:**
1. Run Smart Fill
2. Review all suggestions
3. Do NOT apply yet

**Expected Results:**
- [ ] Preview mode shows proposed changes
- [ ] Schedule not yet modified
- [ ] Can review everything
- [ ] Apply/Cancel options clear

---

### TC-16.5.12: Apply All Suggestions

**Steps:**
1. Run Smart Fill
2. Review suggestions
3. Click "Apply All"

**Expected Results:**
- [ ] All suggestions applied
- [ ] Shifts assigned as suggested
- [ ] Success confirmation shown
- [ ] Schedule is updated

---

### TC-16.5.13: Apply Individual Suggestions

**Steps:**
1. Run Smart Fill
2. For individual shift, click "Apply"
3. Leave others unchanged

**Expected Results:**
- [ ] Can apply suggestions individually
- [ ] Selected shift assigned
- [ ] Other shifts remain unassigned
- [ ] Partial application works

---

### TC-16.5.14: Reject/Modify Suggestions

**Steps:**
1. Run Smart Fill
2. Select different employee than suggested
3. Apply modified selection

**Expected Results:**
- [ ] Can override suggestions
- [ ] Select any qualified employee
- [ ] Override is applied
- [ ] System learns from overrides (optional)

---

### TC-16.5.15: Cancel Smart Fill

**Steps:**
1. Run Smart Fill
2. Click "Cancel" or exit preview

**Expected Results:**
- [ ] Smart Fill cancelled
- [ ] No changes made to schedule
- [ ] Original state preserved
- [ ] Can run again later
