# UAT: Feature 23.3 - Employee Stats Dashboard

## Test Environment Setup

**Prerequisites:**
- Employee account with historical data (shifts, leave, clock records)
- Admin account for tenant settings configuration
- Tenant with configurable stats visibility settings
- Test data spanning multiple periods

**Test URL:** `/dashboard`, `/profile/stats`

---

## Test Cases

### TC-23.3.1: View Employee Personal Dashboard

**Steps:**
1. Log in as an employee
2. Navigate to Dashboard

**Expected Results:**
- [ ] Personal dashboard loads successfully
- [ ] Shows personalized statistics
- [ ] Layout is clear and organized
- [ ] Loading indicators for data

---

### TC-23.3.2: Display Hours Worked

**Steps:**
1. Log in as an employee with clock-in history
2. View dashboard stats

**Expected Results:**
- [ ] Hours worked this week/period shown
- [ ] Hours worked this month shown
- [ ] Total hours calculation accurate
- [ ] Formatted nicely (e.g., 40h 30m)

---

### TC-23.3.3: Display Leave Balance

**Steps:**
1. Log in as an employee
2. View dashboard stats

**Expected Results:**
- [ ] Leave balance displayed per leave type
- [ ] Vacation days remaining shown
- [ ] Sick days remaining shown
- [ ] Clear indication of entitlement vs used

---

### TC-23.3.4: Display Upcoming Shifts

**Steps:**
1. Log in as an employee with scheduled shifts
2. View dashboard

**Expected Results:**
- [ ] Next few shifts displayed
- [ ] Date, time, location shown
- [ ] Easy to read at a glance
- [ ] Link to full schedule

---

### TC-23.3.5: Attendance/Punctuality Stats - Visible

**Steps:**
1. Tenant settings: attendance stats visible to employees
2. Log in as an employee
3. View dashboard

**Expected Results:**
- [ ] Attendance rate shown (e.g., 98%)
- [ ] Punctuality rate shown
- [ ] Clearly labeled stats
- [ ] Historical trend (if available)

---

### TC-23.3.6: Attendance/Punctuality Stats - Hidden

**Steps:**
1. Tenant settings: attendance stats hidden from employees
2. Log in as an employee
3. View dashboard

**Expected Results:**
- [ ] Attendance rate NOT shown
- [ ] Punctuality rate NOT shown
- [ ] No mention of these metrics
- [ ] Other stats still visible

---

### TC-23.3.7: Admin Configure Stats Visibility

**Steps:**
1. Log in as admin
2. Navigate to Settings > Tenant Settings
3. Find employee stats visibility options
4. Toggle attendance/punctuality visibility

**Expected Results:**
- [ ] Stats visibility options available
- [ ] Can enable/disable each stat type
- [ ] Settings save successfully
- [ ] Changes apply immediately

---

### TC-23.3.8: Stats Period Selection

**Steps:**
1. View employee stats dashboard
2. Change period (this week, this month, custom)

**Expected Results:**
- [ ] Period selector available
- [ ] Stats recalculate for selected period
- [ ] Custom date range works
- [ ] Data accurate for period

---

### TC-23.3.9: No Data State

**Steps:**
1. Log in as a new employee with no history
2. View dashboard stats

**Expected Results:**
- [ ] Graceful handling of no data
- [ ] "No data yet" or similar message
- [ ] No errors or broken layout
- [ ] Encouragement to use the system

---

### TC-23.3.10: Hours Calculation Accuracy

**Steps:**
1. Employee with known clock records
2. Manually calculate expected hours
3. Compare with dashboard

**Expected Results:**
- [ ] Dashboard hours match manual calculation
- [ ] Breaks deducted correctly (if applicable)
- [ ] Overtime counted correctly
- [ ] Rounding consistent

---

### TC-23.3.11: Leave Balance Accuracy

**Steps:**
1. Employee with known leave entitlement and used
2. Manually calculate expected balance
3. Compare with dashboard

**Expected Results:**
- [ ] Balance matches entitlement - used
- [ ] Pending leave requests shown separately
- [ ] Calculation matches other reports
- [ ] No discrepancies

---

### TC-23.3.12: Dashboard Performance

**Steps:**
1. View dashboard with substantial historical data
2. Observe load time

**Expected Results:**
- [ ] Dashboard loads quickly (< 3 seconds)
- [ ] Stats load without blocking UI
- [ ] No timeout errors
- [ ] Smooth user experience

---

### TC-23.3.13: Mobile Dashboard View

**Steps:**
1. View employee dashboard on mobile device
2. Review stats display

**Expected Results:**
- [ ] Responsive layout works
- [ ] Stats readable on small screen
- [ ] No horizontal scrolling needed
- [ ] Touch interactions work

---

### TC-23.3.14: Stats Refresh

**Steps:**
1. View dashboard
2. Clock in (in another tab/app)
3. Refresh dashboard

**Expected Results:**
- [ ] Stats update after clock-in
- [ ] No stale data
- [ ] Real-time or near-real-time updates
