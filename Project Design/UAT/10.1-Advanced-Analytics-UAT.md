# UAT: Feature 10.1 - Advanced Analytics & Reports (Premium)

## Test Environment Setup

**Prerequisites:**
- Admin user with premium/enterprise subscription
- Historical data (shifts, clock events, labor costs)
- Multiple locations and departments
- Employee without premium access for gate testing

**Test URL:** `/reports/builder`, `/reports/analytics`

---

## Test Cases

### TC-10.1.1: Feature Gate - Premium Required

**Steps:**
1. Log in as admin without premium subscription
2. Try to access advanced reports

**Expected Results:**
- [ ] Access denied or upgrade prompt shown
- [ ] Feature clearly marked as premium
- [ ] Link to upgrade subscription available
- [ ] Basic reports still accessible

---

### TC-10.1.2: Report Builder Interface

**Steps:**
1. Log in as admin with premium access
2. Navigate to Report Builder
3. Observe interface elements

**Expected Results:**
- [ ] Page loads successfully
- [ ] Drag-and-drop interface available
- [ ] Available data fields shown
- [ ] Report preview area visible

---

### TC-10.1.3: Create Custom Report

**Steps:**
1. In report builder, drag fields to report:
   - Employee name
   - Total hours
   - Date range
2. Set filters (date range, department)
3. Preview report
4. Save report

**Expected Results:**
- [ ] Fields can be dragged to report canvas
- [ ] Filters apply correctly
- [ ] Preview shows expected data
- [ ] Report saves successfully

---

### TC-10.1.4: Save and Name Report

**Steps:**
1. Build a custom report
2. Click Save
3. Enter report name and description
4. Save

**Expected Results:**
- [ ] Save dialog appears
- [ ] Can enter name and description
- [ ] Report saved to saved reports list
- [ ] Can access later

---

### TC-10.1.5: Share Report

**Steps:**
1. Open a saved report
2. Click Share option
3. Share with another admin user
4. Verify other user can access

**Expected Results:**
- [ ] Share option available
- [ ] Can select users to share with
- [ ] Shared user can view report
- [ ] Owner maintains edit rights

---

### TC-10.1.6: Labor Cost Trend Analysis

**Steps:**
1. Navigate to Labor Cost Analytics

3. View trend charts

**Expected Results:**
- [ ] Cost trends displayed graphically
- [ ] Can see week-over-week changes

- [ ] Total costs calculated correctly

### TC-10.1.7: Schedule Efficiency Metrics

2. View metrics

### TC-10.1.8: Scheduled Email Delivery
