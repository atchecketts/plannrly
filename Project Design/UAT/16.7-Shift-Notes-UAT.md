# UAT: Feature 16.7 - Shift Notes & Handover

## Test Environment Setup

**Prerequisites:**
- Admin user account
- Employee user accounts
- Scheduled shifts for testing
- Consecutive shifts for handover testing

**Test URL:** `/shifts/{id}/notes`

---

## Test Cases

### TC-16.7.1: Manager Add Instructions When Creating Shift

**Steps:**
1. Log in as admin
2. Create a new shift
3. Add manager instructions/notes
4. Save the shift

**Expected Results:**
- [ ] Instructions field available on shift form
- [ ] Text can be entered
- [ ] Instructions save with shift
- [ ] Instructions visible on shift details

---

### TC-16.7.2: Manager Edit Shift Instructions

**Steps:**
1. Log in as admin
2. Edit an existing shift
3. Modify the instructions
4. Save changes

**Expected Results:**
- [ ] Can edit existing instructions
- [ ] Changes are saved
- [ ] History of changes maintained (optional)

---

### TC-16.7.3: Employee View Shift Instructions

**Steps:**
1. Log in as employee
2. View assigned shift details
3. Find the instructions section

**Expected Results:**
- [ ] Instructions visible to assigned employee
- [ ] Clear presentation of instructions
- [ ] Cannot modify manager instructions
- [ ] Prominent display

---

### TC-16.7.4: Employee Add Handover Notes

**Steps:**
1. Log in as employee
2. Navigate to current/completed shift
3. Add handover notes
4. Save

**Expected Results:**
- [ ] Handover notes field available
- [ ] Can add text notes
- [ ] Notes save successfully
- [ ] Timestamp recorded

---

### TC-16.7.5: Handover Notes During Shift

**Steps:**
1. Clock in for shift
2. Add handover note during shift
3. Save note

**Expected Results:**
- [ ] Can add notes during shift
- [ ] Multiple notes can be added
- [ ] Timestamp for each note
- [ ] Notes are editable until shift ends

---

### TC-16.7.6: View Handover From Previous Shift

**Steps:**
1. Complete a shift with handover notes
2. Start the next shift (same location/role)
3. View handover from previous shift

**Expected Results:**
- [ ] Previous shift handover visible
- [ ] Shows who wrote the notes
- [ ] Shows when notes were written
- [ ] Easy to access at shift start
