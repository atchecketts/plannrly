# UAT: Feature 29.4 - Employee Personal Insights

## Test Environment Setup

**Prerequisites:**
- Employee user with work history (shifts, attendance)
- At least 3 months of shift data for the employee
- Leave balance configured
- Time clock data available
- Employee with various shift types

**Test URL:** `/dashboard/insights`, `/my-insights`

---

## Test Cases

### TC-29.4.1: Access Personal Insights

**Steps:**
1. Log in as employee
2. Navigate to Dashboard
3. Find "My Insights" section or link
4. Click to view full insights

**Expected Results:**
- [ ] Insights section visible on dashboard
- [ ] Link to detailed insights available
- [ ] Page loads without errors
- [ ] Personal data is displayed

---
