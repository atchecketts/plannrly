# UAT: Feature 22.3 - Internationalization

## Test Environment Setup

**Prerequisites:**
- Admin user with locale settings access
- Browser with language preferences
- Knowledge of test phrases in supported languages
- Test accounts in different timezones

**Supported Languages:** EN, ES, FR, DE, IT, PT

**Test URL:** `/settings/locale`, `/profile/preferences`

---

## Test Cases

### TC-22.3.1: View Available Languages

**Steps:**
1. Log in as an admin
2. Navigate to Settings > Locale/Language
3. View available language options

**Expected Results:**
- [ ] All 6 languages listed (EN, ES, FR, DE, IT, PT)
- [ ] Current language highlighted
- [ ] Language names shown in their native form
- [ ] Easy selection interface

---

### TC-22.3.2: Set Tenant Default Locale

**Steps:**
1. Log in as an admin
2. Navigate to Settings > Locale
3. Set default tenant language to Spanish (ES)
4. Save changes

**Expected Results:**
- [ ] Default language saved
- [ ] Confirmation message displayed
- [ ] New users default to Spanish
- [ ] Setting reflected in tenant settings

---

### TC-22.3.3: User Language Preference Override

**Steps:**
1. Tenant default is Spanish
2. Log in as an employee
3. Navigate to Profile > Preferences
4. Set personal language to French (FR)
5. Save and refresh

**Expected Results:**
- [ ] User can override tenant default
- [ ] Interface changes to French
- [ ] Personal preference saved
- [ ] Does not affect other users

---

### TC-22.3.4: Language Switching - English to Spanish

**Steps:**
1. Set language to English
2. Navigate through the app
3. Switch language to Spanish
4. Navigate through the same pages

**Expected Results:**
- [ ] All UI text changes to Spanish
- [ ] Navigation items translated
- [ ] Button labels translated
- [ ] No untranslated strings visible

---

### TC-22.3.5: Language Switching - All Languages

**Steps:**
1. For each language (ES, FR, DE, IT, PT):
   - Set as current language
   - Navigate to Dashboard, Schedule, Settings
   - Check for untranslated text

**Expected Results:**
- [ ] Spanish translations complete
- [ ] French translations complete
- [ ] German translations complete
- [ ] Italian translations complete
- [ ] Portuguese translations complete

---

### TC-22.3.6: Date Formatting by Locale

**Steps:**
1. Set language to English (US)
2. View a date (e.g., shift date)
3. Switch to German
4. View the same date

**Expected Results:**
- [ ] English shows MM/DD/YYYY or similar
- [ ] German shows DD.MM.YYYY
- [ ] Each locale uses appropriate format
- [ ] Dates are never ambiguous

---

### TC-22.3.7: Time Formatting by Locale

**Steps:**
1. Set language to English (US)
2. View a time (e.g., shift time)
3. Switch to French
4. View the same time

**Expected Results:**
- [ ] English may show 12-hour format (3:00 PM)
- [ ] French shows 24-hour format (15:00)

- [ ] Times are accurate
