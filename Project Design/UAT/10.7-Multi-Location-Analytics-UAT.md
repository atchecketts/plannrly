# UAT: Feature 10.7 - Multi-Location Analytics (Enterprise)

## Test Environment Setup

**Prerequisites:**
- Admin user with enterprise subscription
- Multiple locations with data
- Historical schedule and labor data per location
- Variety of performance metrics available

**Test URL:** `/analytics/multi-location`

---

## Test Cases

### TC-10.7.1: Feature Gate - Enterprise Required

**Steps:**
1. Log in as admin with premium (not enterprise)
2. Try to access multi-location analytics

**Expected Results:**
- [ ] Access denied or upgrade prompt shown
- [ ] Feature marked as Enterprise
- [ ] Single-location analytics may work

---

### TC-10.7.2: View Multi-Location Dashboard

**Steps:**

**Expected Results:**

- [ ] Shows all locations
