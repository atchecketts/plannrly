# UAT: Feature 2.7 - Print Schedule

## Test Environment Setup

**Prerequisites:**
- One admin user
- Published shifts for multiple employees
- Access to a printer (physical or PDF)
- Schedule with various locations and departments

**Test URLs:**
- Schedule: `/schedule`
- Print Preview: `/schedule/print`

---

## Test Cases

### TC-2.7.1: Access Print Option from Week View

**Steps:**
1. Log in as an admin
2. Navigate to schedule week view
3. Look for "Print" button or option

**Expected Results:**
- [ ] Print button/icon is visible
- [ ] Located in toolbar or actions area
- [ ] Click opens print options or preview

---

### TC-2.7.2: Print Preview Week View

**Steps:**
1. Log in as an admin
2. Navigate to schedule week view
3. Click "Print"
4. View print preview

**Expected Results:**
- [ ] Print preview loads
- [ ] Shows full week schedule
- [ ] Layout optimized for landscape
- [ ] All shifts visible and readable

---

### TC-2.7.3: Print Preview Day View

### TC-2.7.4: Print with Location Filter

### TC-2.7.8: Print with Employee Contact Details

**Steps:**
1. Log in as an admin
2. Open print options
3. Check "Include contact details"
4. Preview or print

**Expected Results:**
- [ ] Employee phone numbers shown
- [ ] Contact info in readable format
- [ ] Privacy warning may be shown
- [ ] Useful for posting schedules

---

### TC-2.7.9: Print without Contact Details

**Steps:**
1. Log in as an admin
2. Open print options
3. Uncheck "Include contact details"
4. Preview or print

**Expected Results:**
- [ ] Phone numbers NOT shown
- [ ] Only names and shift times
- [ ] More privacy-friendly output
